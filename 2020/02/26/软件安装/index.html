软件安装

Posted on 2020-02-26 Edited on 2020-06-25

web软件安装

  • JDK安装+eclips

    1. 推荐下载稳定的JDK8
    2. 系统环境配置三个
      JAVA_HOME : jdk目录(修改JDK版本)
      CLASSPATH: .%JAVA_HOME%\lib
      path添加 .%JAVA_HOME%\bin
    3. Eclips 根据自己的需求安装

  • intellij IDE

    码f8xv

    1. 安装IDE,将JetbrainsCrack.jar复制到bin
    2. 选择免费试用，之后选择help->Edit custom VM….
    3. 根据自己地址添加-javaagent:F:\idea\IntelliJ IDEA 2019.1.2\bin\JetbrainsCrack.jar
    4. 重启，help->register,选择激活码激活
    5. help->about查看结果

  • navicat

    码263k

    1. 安装
    2. 打开注册机选择exe文件patch
    3. 打开navicat注册，generate Serial Keygen，复制生成码点注册->手动->复制请求码generate激活码

  • tomcat

    安装后记录

    1. …\bin\startup.bat启动

    2. …\conf\server.xml 配置端口和http get传递中文乱码问题

    3. …conf\context.xml
      <context reloadable="true">

    4. …conf\tomcat-users.xml 配置管理账号

      <role rolename = "manage-gui"/>
      <user username="tomcat" password="tomcat" roles="manage-gui"/>

  • mysql

    1. 添加环境变量 path添加...\bin地址
    2. mysql目录下新建my.ini

  1. 进入bin目录，管理员权限启动cmd输入
    mysqld --initialize --console
    等下出现的root@localhost：后面的是随机生成的数据库初始密码，执行完后输入 

    mysqld --install
net start mysql

  2. 登录修改密码 

    mysql -uroot -p上面的密码
alter user root@localhost identified by '新密码';
